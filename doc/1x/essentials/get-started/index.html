Get started

Instalar a CLI

Nimble possui uma CLI (Command Line Interface) essencial para que tudo funcione perfeitamente. Com ele é possível fazer desde a criação de um novo projeto até a criação de páginas. Veja a documentação completa dele clicando aqui.

Para instala-la globalmente, execute em seu terminal a linha abaixo: 
            
$ npm i @nimble-ts/cli -g

⚠️ Problemas ao instalar? Veja mais detalhes aqui.

Criar um projeto

01 - Run in your terminal:
            
$ nb
02 - Ele lhe perguntará se quer criar um projeto Nimble:
            
$ ? Do you want create new Nimble project? (Y/n)
03 - Então, coloque o nome do seu projeto e confirme:
            
? What is name of the your project? (ex.: my-project)
04 - Pronto, um diretório com o nome dado será criado com uma estrutura de pastas como esta abaixo:
            
public
├─ assets
├─ favicon.ico
└─ index.html
src
├─ app
├─ environments
├─ scss
├─ main.ts
└─ style.scss
nimble.json
package.json
tsconfig.json

⚠️ Entenda toda a estrutura do projeto clicando aqui.

05 - Agora, basta pedir para a CLI do Nimble iniciar um servidor de desenvolvimento para ver o site rodando, execute a linha abaixo:
            
nb serve

⚠️ Saiba mais sobre a Nimble CLI clicando aqui.
