Sobre

Diretivas

As diretivas são classes que podem conter qualquer regra que se queira aplicar a partir de um elemento HTML renderizado. As diretivas possuem seletores (que são atributos ou propriedades) únicos que as identificam, para que sejam inseridas em elementos dos templates de páginas ou diálogos e possam ser interpretadas e executadas.

Se você não sabe como criar uma Diretiva para o seu projeto, veja como clicando aqui.

A estrutura

A estrutura de uma diretiva é muito simples, possuindo um decorador chamado @PrepareDirective que fica acima da declaração da classe e ela também deve estender a classe Directive.

No decorador citado acima, é enviado um objeto contendo até 3 propriedades para definir a diretiva, cada um delas tem sua função, veja:

❯ selector (obrigatória) é a propriedade que definirá o nome de identificação da diretiva, que pode ser uma string ou um array de strings.

❯ inputs (opcional) é a propriedade que define se a diretiva terá entradas (dados), através de outros seletores.

❯ selector (opcional) é a propriedade que define se a diretiva terá saídas (dados), através de outros seletores (sempre deve ser entre parenteses, por exemplo (onChange)).

Veja o exemplo de uma diretiva que, onde sua função será focar em um campo textarea quando a tela for renderizada:

import { PrepareDirective, Directive } from '@nimble-ts/core/directives';

@PrepareDirective({
    selector: 'force-focus',
    inputs: [],
    outputs: []
})
export class ForceFocusDirective extends Directive {

    constructor() {
        super();
    }

    public onRender(): void {
        this.element.focus();
    }

    public onChange(): void {
    }

    public onDestroy() {
    }
}

Como utilizar

Após ter uma diretiva criada, veja abaixo os simples passos que devem ser feitos para que ela funcione perfeitamente.

Importar no projeto

A primeira coisa a se fazer é importa-la na raiz do projeto, na inicialização do Nimble, dentro de directives, veja um exemplo:

import { NimbleApp } from '@nimble-ts/core';
import { ROUTES } from './app/routes';
import { ForceFocusDirective } from './app/directives/force-focus.directive';

NimbleApp.config({
    routes: ROUTES,
    directives: [
        ForceFocusDirective
    ],
    providers: []
}).start()
Utilizar no template

Como foi introduzido anteriormente, toda diretira deve possuir um seletor único, o qual será utilizado nos templates conforme a necessidade. Veja o exemplo da diretive que foi mencionado na introdução, onde é utilizada no input de uma página de login:

<div class="login-page">
    <div class="container">
        <input type="email" placeholder="E-mail" force-focus />
        <input type="password" placeholder="Password" />
    </div>
</div>
